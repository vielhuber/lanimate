lanimate is a small library that animates appearing dom elements.

## installation

use it as a module:

```
npm install lanimate
```

```js
import lanimate from 'lanimate';
```

or as a ready-to-go bundle:

```html
<script src="lanimate.min.js"></script>
```

## usage

```js
const l = new lanimate();
l.init();
```
